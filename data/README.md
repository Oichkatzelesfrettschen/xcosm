# Data Directory

This directory contains raw and processed data for the COSMOS project.

## Raw Data (`data/raw/`)

### Cosmology & Supernovae
*   **Pantheon+ SNe Ia:** `Pantheon+SH0ES.dat` (1702 supernovae from Pantheon+SH0ES collaboration)
*   **Light Curve Parameters:** `lcparam_full_long.txt` (Full light curve fit parameters)
*   **DESI BAO:** `bao_data/` (Cloned from CobayaSampler/bao_data)

### LHC & Particle Physics (`data/raw/lhc_data/`)
*   `cms_small_system_v2.csv`: CMS Small-Scale QGP flow (v2 vs Nch)
*   `atlas_oo_quenching_2025.csv`: ATLAS O-O Jet Quenching (RAA)
*   `w_boson_mass_2025.csv`: Precision W-mass measurements (CMS, ATLAS, CDF)

### Gravitational Waves (`data/raw/gw_sensitivity/`)
*   `LISA_sensitivity.dat`: LISA sensitivity curve (Source: Zenodo/Robson et al.)
*   `ET-0000A-18.txt`: Einstein Telescope (ET-D) sensitivity curve (Source: ET Consortium)

### Nuclear Physics (`data/raw/nuclear/`)
*   `mass_1.mas20`: AME2020 Atomic Mass Evaluation (Source: ANL/IAEA) - *Note: Direct download may require manual mirror check.*

### Asteroseismology
*   `wd_pulsation_periods_table.csv`: White Dwarf pulsation periods.

## Processed Data (`data/processed/`)
Generated by the simulation engines and analysis scripts (regeneratable).
*   `*.npz`: Numpy compressed archives of simulation results.
*   `*.json`: JSON exports of model states.

## Output Plots (`output/plots/`)
Generated visualization outputs (regeneratable, excluded from git).
*   `pantheon_bifurcation_plots.png`: Pantheon+ bifurcation analysis results
*   `pantheon_validation.png`: Data validation visualizations
*   Additional plots generated by analysis scripts

## References (`data/references/`)
*   `references.bib`: BibTeX database of cited literature.
